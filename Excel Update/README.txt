The solution works by employing Powershell to download the excel file from the 
Sharepoint library then run the c# code in the project to change the connection and then
upload back to SharePoint using powershell again.
The OpenXML application installs the OpenXML 2.0 library in your machine. The project has 
this as one of it's dependencies. Plus the machine must have .NET 4.5 installed as well.
